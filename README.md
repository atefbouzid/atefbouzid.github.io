## Customization

### Personal Information

1. **Name and Title**: Update in `index.html`
   - Line 12: Update `<title>` tag
   - Line 30: Update nav brand name
   - Line 40: Update hero title

2. **About Section**: Edit the about text in `index.html` (lines 75-85)

3. **Skills**: Modify the skills in `index.html` (lines 95-120)

4. **Projects**: Update project cards in `index.html` (lines 130-185)
   - Replace placeholder projects with your actual projects
   - Update project links (GitHub, live demos)

5. **Contact Information**: Update in `index.html` (lines 195-210)
   - Email address
   - LinkedIn profile URL
   - GitHub username
   - Social media links in footer

### Styling

- **Colors**: Modify CSS variables in `styles.css` (lines 5-13)
- **Fonts**: Change the Google Fonts import in `index.html` (line 9)

### Contact Form

The contact form currently shows an alert. To make it functional:

1. Use a service like [Formspree](https://formspree.io/), [EmailJS](https://www.emailjs.com/), or
2. Set up a backend API endpoint
3. Update the form submission handler in `script.js` (lines 60-72)

## Deployment

### GitHub Pages

1. Push your code to a GitHub repository
2. Go to repository Settings ‚Üí Pages
3. Select the branch (usually `main`) and folder (`/root`)
4. Your site will be available at `https://yourusername.github.io`

### Other Hosting Options

- **Netlify**: Drag and drop the folder or connect your Git repository
- **Vercel**: Import your Git repository
- **Any static hosting service**: Upload the files via FTP
